# formation-sparks-copilot-19092025

## Projet Game Session API

Ce repository contient un projet d'API web .NET 8 pour la gestion des sessions de jeux vidéo, développé dans le cadre de la formation Sparks Copilot.

### Structure du projet

- `GameSessionApi/` - API web .NET 8 avec endpoints minimaux pour la gestion des sessions de jeu

### Fonctionnalités réalisées

✅ **Projet .NET 8 LTS** - Utilisation de la dernière version LTS de .NET  
✅ **API Web minimale** - Implémentation avec les Minimal APIs d'ASP.NET Core  
✅ **Gestion des sessions de jeux vidéo** - CRUD complet pour les sessions  
✅ **Documentation Swagger** - Interface de documentation automatique  
✅ **Recherche avancée** - Filtrage par joueur et par jeu  
✅ **Calcul automatique** - Durée de session calculée automatiquement  

### Démarrage rapide

```bash
cd GameSessionApi
dotnet run
```

L'API sera disponible sur `http://localhost:5126` avec la documentation Swagger sur `/swagger`.

Consultez le [README détaillé](GameSessionApi/README.md) dans le dossier du projet pour plus d'informations.
